# Neovim shortcuts
## tab switch
```
nnoremap <C-N>      :bnext<CR>
nnoremap <C-Right>  :bnext<CR>
nnoremap <C-Left>   :bprev<CR>
```
## Fzf
```
nnoremap <C-P>      :FZF<CR>
```
## Grep
```
nnoremap <C-F>      :Rg<CR>
```
## cmp setup
```
nnoremap <C-F>      :Rg<CR>
```
